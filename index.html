YAZILIMLAR

HTML5

HTML5'de yetkinim ve web sayfalarını semantik yapılara uygun olarak oluşturabiliyorum. Ayrıca, form ve video öğeleri gibi temel etiketleri kullanarak zengin içerikler sunabiliyorum.

CSS3

CSS3'te deneyimliyim ve web sayfalarını estetik ve kullanıcı dostu hale getirebiliyorum. Responsive tasarım ve animasyonlar gibi ileri düzey teknikleri kullanarak kullanıcı deneyimini iyileştirebiliyorum.

JavaScript

JavaScript ile web sayfalarını etkileşimli hale getirebiliyorum. AJAX istekleri ve modern JavaScript özelliklerini kullanarak dinamik ve kullanıcı dostu uygulamalar geliştirmeye çalışıyorum.

Python

Python'da deneyim sahibiyim ve temel syntaxı kavradım. Python ile veri analizi, web geliştirme ve otomasyon gibi çeşitli alanlarda uygulamalar geliştirebiliyorum.

Java

Java'da yetkinim ve nesne yönelimli programlama konseptlerini iyi anlıyorum. Java ile masaüstü uygulamaları ve backend sistemleri geliştirebiliyorum.

React.js

React.js geliştirmekteyim. Component bazlı yapı ve tek yönlü veri akışı gibi temel prensipleri kullanarak kullanıcı dostu arayüzler oluşturabiliyorum.

HAKKIMDA

Merhaba, ben Ömer Körükmez. 21 yaşındayım ve 09.06.2003 tarihinde Malatya'nın Kuluncak ilçesinde doğdum. Yazılım dünyasına olan tutkum küçük yaşlardan itibaren başladı. İlk programlama deneyimlerimle bilgisayarları daha derinlemesine anlamaya başladım.

Şu anda İnönü Üniversitesi'nde Yazılım Mühendisliği programının ikinci sınıfında öğrenciyim. Üniversitedeki eğitimim boyunca, yazılım dünyasının farklı alanlarını keşfetme fırsatı buldum. Hem teorik derslerde hem de pratik projelerde kendimi geliştirdim.

Kod yazmayı ve yazılım işleri ile uğraşmayı çok seviyorum. Özellikle Java, C ve JavaScript gibi dillerde uzmanlaşmaya çalışıyorum. Java dilinde ileri seviyede programlama ve geliştirme yapabiliyorum. Ayrıca web geliştirme konusunda da deneyimlerim var ve sürekli yeni teknolojileri öğrenmeye açığım.

Gelecekte yazılım alanında başarılı bir kariyere sahip olmayı hedefliyorum. Yazılımın hayatımızdaki rolünü daha da derinlemesine anlamak ve insanların hayatını kolaylaştırmak için çalışmalarıma devam ediyorum.
